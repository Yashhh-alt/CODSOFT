# Movie Rating Prediction

## Overview

This project predicts the rating of a movie using Machine Learning based on features such as Genre, Director, and Actors. A Random Forest Regression model is trained on historical movie data and integrated with a Tkinter GUI.

## Features

* Movie rating prediction using Machine Learning
* User-friendly graphical interface
* Random Forest Regression model
* Real-time rating estimation
* Data preprocessing and feature encoding

## Technologies Used

* Python
* Tkinter
* Pandas
* Scikit-Learn

## Dataset

IMDb Movies India Dataset

### Dataset Attributes

* Name
* Year
* Duration
* Genre
* Rating
* Votes
* Director
* Actor 1
* Actor 2
* Actor 3
