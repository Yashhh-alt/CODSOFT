# Titanic Survival Prediction

## Overview

This project uses Machine Learning to predict whether a passenger on the Titanic would survive based on passenger details.

## Features

* Modern GUI using Tkinter
* Random Forest Classification Model
* Survival Prediction
* User-friendly Interface

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Tkinter

## Dataset

Titanic-Dataset.csv

## Installation

pip install -r requirements.txt

## Run

python titanic.py
